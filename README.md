# World Happiness Report 2019 – Data Analysis in R

## Overview
This project analyzes the World Happiness Report 2019 dataset using R.
The goal is to understand how factors such as GDP, social support,
health, and corruption influence happiness scores across countries.

## Dataset
- Source: World Happiness Report 2019 (Kaggle)
- Rows: 156 countries
- Key variables:
  - Happiness Score
  - GDP per capita
  - Social support
  - Healthy life expectancy
  - Perceptions of corruption

## Analysis Performed
- Exploratory data analysis
- Relationship between GDP and happiness
- Impact of corruption perception on happiness
- Happiness level classification (High / Medium / Low)
- Identification of top 10 happiest countries

## Tools & Libraries
- R
- dplyr
- ggplot2
- pacman

## Files
- `world-happiness-analysis.R` – Main analysis script
- `2019.csv` – Dataset
- `top_10_happy.csv` – Output data
- `report.Rmd` – Reproducible analysis report
